# React Marusya App

Современный одностраничный онлайн-кинотеатр для портфолио: можно искать фильмы, изучать подборки по жанрам, смотреть карточки с трейлерами и держать свой список избранного после авторизации. Проект создан на React и TypeScript с кастомной сборкой на Webpack.

## Возможности
- Главная страница с героем и блоком «топ фильмов».
- Поиск по каталогу прямо из шапки сайта.
- Каталог жанров и отдельные страницы подборок по каждому жанру.
- Детальная страница фильма с описанием, трейлером и похожими подборками.
- Авторизация и регистрация через модальное окно, работа с сервером через cookies.
- Личный кабинет с настройками профиля и списком избранных фильмов.

## Сервер и API
- Клиент работает с публичным API `https://cinemaguide.skillbox.cc` (конфигурация в `src/services/api.ts`).
- Swagger-описание лежит в `swagger.json`; по нему сгенерирован типизированный клиент (`src/api/Api.ts`).
- Основные группы методов: `auth` (логин/регистрация), `favorites` (добавление/удаление/получение избранного), `movies` (каталог, жанры, поиск, детали).
- Запросы отправляются с включёнными cookie (`credentials: 'include'`), чтобы сервер мог аутентифицировать пользователя.

## Технологии
- React 19, TypeScript, React Router 7.
- Формы и валидация: React Hook Form + Zod.
- Работа с API: Axios + сгенерированный Swagger-клиент.
- Стилизация: SCSS + modern-normalize, компоненты UI иконки в SVG.
- Сборка: Webpack 5, поддержка Hot Module Replacement в разработке.

## Как запустить локально
1. Установите зависимости: `npm install`.
2. Запустите дев-сервер: `npm start`. По умолчанию приложение откроется на `http://localhost:3000` с поддержкой HMR.
3. Продакшен-сборка: `npm run build:prod` (или `build:mobile` / `build:desktop` под соответствующую платформу).
4. Проверка типов: `npm run typecheck`.

## Скрипты
- `npm start` — дев-сервер Webpack с HMR.
- `npm run build:dev` — дев-сборка без запуска сервера.
- `npm run build:prod` — продакшен-сборка.
- `npm run build:gh` — продакшен-сборка с публичным путём для GitHub Pages.
- `npm run build:mobile` / `npm run build:desktop` — оптимизированные сборки под разные платформы.
- `npm run typecheck` — проверка типов TypeScript.

## Структура
- `src/pages` — страницы (главная, жанры, подборки по жанрам, карточка фильма, аккаунт).
- `src/features` — функциональные блоки (авторизация, поиск, подборки и т.д.).
- `src/layout` — общие элементы макета (шапка, подвал).
- `src/services` — конфигурация API-клиента.
- `src/api` — сгенерированные типы и методы для работы с сервером.
- `config` — конфигурации сборки Webpack и dev-server.

Проект оформлен как портфолио: акцент на чистой структуре, типизации и удобстве локального запуска.
